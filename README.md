# RPG Worldbuilder Helper

A static fantasy-journal web app that asks worldbuilding questions for an RPG campaign or book setting.

The app lets you:

- create a new world,
- answer questions by topic,
- save progress in the browser,
- export both a readable PDF and a JSON save file,
- import the JSON save file,
- import PDFs generated by this app,
- review answered, unanswered, or all questions,
- see completion progress across all topics.

## Publishing with GitHub Pages

After these files are uploaded:

1. Open the repository on GitHub.
2. Go to **Settings**.
3. Go to **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose branch: `main`.
6. Choose folder: `/ (root)`.
7. Save.

The site should become available at:

`https://gamullus.github.io/rpg-worldbuilder-helper/`

If the link shows a 404, wait a minute and refresh, or check that GitHub Pages is enabled from the repository settings.

## Notes

Browser save is local to the current browser/device. For long-term saves or moving between devices, use the exported JSON file. The PDF is readable and can also be imported if it was generated by this app, but JSON is the safest save format.
